# Task-Manager-App
📌 Overview

This is a simple task management web application that allows users to add, complete, and delete tasks. The project was built to practice core JavaScript concepts and improve front-end development skills.

🚀 Features
Add new tasks
Mark tasks as completed
Delete tasks
Persistent data using localStorage (tasks remain after page refresh)
Dynamic UI updates using JavaScript
🛠️ Technologies Used
HTML
CSS
JavaScript
💡 What I Learned
DOM manipulation using JavaScript
Handling user input and events
Managing application state
Using localStorage for data persistence
Structuring a small front-end project
🔧 Future Improvements
Add task filtering (All / Completed / Active)
Improve UI/UX design
Add edit task functionality
Convert the app to React
Add backend support for storing tasks
▶️ How to Run the Project
Clone the repository
Open the folder
Run index.html in your browser
